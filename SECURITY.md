# Security Policy

## Supported Versions
The main branch is actively maintained. Please update to the latest commit before reporting issues.

## Reporting a Vulnerability
If you discover a vulnerability, please email security@xrpbloomberg.invalid with:
- A proof of concept or clear reproduction steps.
- Impact assessment and potential mitigations.
- Your preferred disclosure timeline.

We aim to acknowledge new reports within 72 hours. Do not open public issues for security problems until a coordinated disclosure plan is in place.
