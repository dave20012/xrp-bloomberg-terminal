# Agent Instructions

This repository currently has no project-specific agent guidance. Use the following practices when working here:

- If the browser tooling (a.k.a. Firelight) is available and external browsing is permitted, feel free to use it to gather design ideas and improvement inspiration for the dashboard. Always summarize any external findings in your work so future contributors understand the rationale.
- When browser access is unavailable, continue using local resources (code, tests, docs) to propose design improvements.
- Follow the general coding standards provided by the system instructions. Avoid adding try/catch around imports.

These instructions apply to the entire repository unless overridden by a nested `AGENTS.md` file.
